# 🏨 Hotel Management System

A simple **Hotel Management System** built using **Python** and **Object-Oriented Programming (OOP)** concepts. This project simulates basic hotel operations for both managers and customers through a command-line interface.

## 📌 Project Overview

This project was developed to practice Python programming fundamentals and object-oriented programming concepts such as classes, objects, methods, constructors, and user interaction.

The system provides two types of users:

* **Manager**
* **Customer**

---

## ✨ Features

### 👨‍💼 Manager Module

* Secure login authentication
* Add new rooms
* Edit room details
* Delete existing rooms

### 👤 Customer Module

* View hotel services
* View available room types
* Room booking (Check-In)
* Select room category
* Calculate total bill based on stay duration
* Date validation using Python's `datetime` module
